# Projet-annuel
# Reconnaissance d’instruments de musique à partir d’images

## Problématique

Développer un modèle capable de reconnaître automatiquement les instruments de musique à partir d’images.

## Dataset choisi

- **Nom** : Musical Instruments Image Classification
- **Source** : Kaggle
- **Lien** : https://www.kaggle.com/datasets/gpiosenka/musical-instruments-image-classification
## Organisation du dataset

Le dataset est déjà organisé en trois dossiers principaux :
- `train/` : images pour l’entraînement
- `valid/` : images pour la validation
- `test/` : images pour le test

Chaque dossier contient un sous-dossier par instrument (ex : `guitar/`, `piano/`, etc.).

## Pistes de constitution du dataset

- Utilisation d’un dataset public existant pour garantir la diversité et la qualité des images.
- Possibilité d’augmenter le dataset plus tard si besoin.
